## рҹҺҜ Tentang Project

**StudyMate AI** mendemonstrasikan implementasi penuh pipeline RAG dari nol tanpa framework berat вҖ” hanya Python, ChromaDB, dan OpenRouter API.

Upload materi kuliah, buku, modul, atau dokumen apapun вҶ’ AI akan memahami isinya dan siap menjawab pertanyaan, meringkas, serta membuat soal quiz secara otomatis.

> **Kenapa project ini penting untuk portfolio AI Engineer?**
> RAG adalah skill paling dicari di industri AI saat ini. Project ini membuktikan pemahaman end-to-end: document ingestion вҶ’ chunking вҶ’ embedding вҶ’ vector search вҶ’ generation.

**Kenapa RAG lebih baik dari langsung tanya LLM?**

- вң… Jawaban berdasarkan dokumen spesifik lo, bukan data training model
- вң… Tidak hallucinate karena ada konteks nyata
- вң… Bisa dipakai untuk dokumen privat/internal
- вң… Lebih hemat token daripada kirim seluruh dokumen

## рҹ“– Cara Pakai

### 1. Setup
- Buka tab **вҡҷпёҸ Setup**
- Masukkan **OpenRouter API Key** (gratis di openrouter.ai)
- Upload file **`.pdf`** atau **`.txt`**
- Klik **рҹҡҖ Proses Dokumen** вҖ” dokumen akan di-chunk dan disimpan ke ChromaDB

### 2. Tanya Jawab (RAG)
- Buka tab **рҹ’¬ Tanya Jawab**
- Ketik pertanyaan tentang isi materi
- AI akan mencari bagian relevan di ChromaDB, lalu menjawab berdasarkan konteks tersebut

### 3. Buat Ringkasan
- Buka tab **рҹ“қ Ringkasan**
- Klik **вңЁ Buat Ringkasan**
- AI menganalisis bagian awal, tengah, dan akhir dokumen untuk membuat ringkasan terstruktur

### 4. Generate Quiz
- Buka tab **рҹ§  Quiz**
- Pilih jumlah soal (3вҖ“10)
- Klik **рҹ§  Generate Quiz**
- Soal dan jawaban di-generate otomatis dari isi materi
- Klik **рҹ’Ў Lihat Jawaban** per soal untuk reveal jawaban

### Document Chunking
```python
# Dokumen dipotong jadi chunk 800 kata dengan overlap 100 kata
# Overlap mencegah informasi terpotong di tengah kalimat
```

### Vector Search
```python
# Query dikonversi ke vector вҶ’ dicari top-3 chunk paling relevan
# Menggunakan cosine similarity di ChromaDB

    query_embeddings=[query_embedding],
    n_results=3
```
